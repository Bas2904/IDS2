Short description:
This program is a simple speech recognizer that takes user input from microphone, and returns an answer.
The data produced by the bot, is then saved to the disk in a .txt file.


The following packages will need to be installed on your PC, in order for the program to work.
Program is built in Python 3.9.7 64bit

gTTS
SpeechRecognition
PyAudio
pygame
selenium
